# wrapper_minecraft-cookbook

This is a wrapper cookbook for my private minecraft server.

## Supported Platforms

- Ubuntu

## Usage

### wrapper_minecraft::default

Include `wrapper_minecraft` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[wrapper_minecraft::default]"
  ]
}
```
